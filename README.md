# GitHub Stargazer Explorer
### How to run
```sh
npm install
npm start
```
When using Yarn
```sh
yarn install
yarn run
```

### Environment variables
```sh
REACT_APP_GITHUB_TOKEN=(TOKEN)
REACT_APP_GITHUB_ENDPOINT=https://api.github.com/graphql
```
